# travelfootprint
# 旅行足迹地图社交网站

一个基于 Next.js + Leaflet 的旅行足迹地图社交网站，支持用户记录旅行足迹、上传旅行照片和视频、查看好友的旅行地图。

## 功能特性

### 1. 登录与用户
- 支持微信登录
- 获取用户昵称、头像
- 显示已注册本站的微信好友列表

### 2. 世界地图
- 陆地：白色
- 海洋：蓝色
- 国家边界：黑色实线
- 国家内部行政区划：灰色实线
- 支持鼠标滚轮、双指触控缩放
- 最小缩放：一屏显示完整世界地图
- 双击城市区域进入城市子页面

### 3. 城市子页面
- 上传图片（JPG）或视频（MP4，时长 < 30 秒）
- 添加文字描述
- 选择公开 / 仅自己可见 / 仅好友可见
- 查看已上传内容
- 支持点赞、评论

### 4. 地图上色规则
- 上传内容后，城市区域自动填充颜色
- 填充色取自该城市第一张图片的主色调

### 5. 社交功能
- 右侧常驻好友头像列表
- 查看好友的世界地图
- 查看好友的公开内容

### 6. 额外功能
- 导出个人世界地图图片
- 设置地图可见范围

## 技术栈

- **前端框架**：Next.js 14
- **地图服务**：Leaflet
- **样式**：Tailwind CSS
- **语言**：TypeScript

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 启动生产服务器

```bash
npm start
```

## 注意事项

1. 本项目使用模拟数据，实际部署时需要：
   - 集成微信登录 SDK
   - 实现后端 API
   - 配置云存储服务
   - 获取真实的城市 GeoJSON 数据

2. 地图样式可根据需要在 `globals.css` 中调整

3. 响应式设计已适配 PC、手机、平板
