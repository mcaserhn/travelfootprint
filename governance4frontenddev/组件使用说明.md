# 组件使用说明

> 本文档仅说明组件使用方式，
> 不定义任何规则。
> 组件约束与边界以《组件基线说明.md》为准。


## 概述

本文档说明项目中统一组件的使用方式，确保团队成员按照规范使用组件，保持代码的一致性和可维护性。

## Button 组件

### 位置
`components/ui/Button.tsx`

### 功能
提供统一的按钮样式，支持多种变体，确保品牌一致性。

### 导入方式
```tsx
import Button from '../components/ui/Button';
```

### 使用方式

#### 基本使用
```tsx
<Button variant="primary">
  按钮文本
</Button>
```

#### 变体
Button 组件支持三种变体：

1. **primary** - 主按钮（橙色背景）
   ```tsx
   <Button variant="primary">
     主要操作
   </Button>
   ```

2. **dark** - 深色按钮（深蓝色背景）
   ```tsx
   <Button variant="dark">
     次要操作
   </Button>
   ```

3. **outline** - 轮廓按钮（橙色边框）
   ```tsx
   <Button variant="outline">
     辅助操作
   </Button>
   ```

#### 与 Link 结合使用
```tsx
import Link from 'next/link';

<Link href="/contact" passHref>
  <Button variant="primary">
    联系我们
  </Button>
</Link>
```

#### 自定义类名
```tsx
<Button variant="primary" className="w-full">
  全宽按钮
</Button>
```

#### 事件处理
```tsx
<Button 
  variant="primary" 
  onClick={() => console.log('按钮被点击')}
>
  点击我
</Button>
```

## Card 组件

### 位置
`components/ui/Card.tsx`

### 功能
提供统一的卡片样式，包含头部、主体和底部三个子组件，用于展示内容区块。

### 导入方式
```tsx
import Card, { CardHeader, CardBody, CardFooter } from '../components/ui/Card';
```

### 使用方式

#### 基本使用
```tsx
<Card>
  <CardHeader>
    <h3 className="text-xl font-title font-bold">卡片标题</h3>
  </CardHeader>
  <CardBody>
    <p className="text-aden-dark-gray font-body">卡片内容...</p>
  </CardBody>
  <CardFooter>
    <Button variant="primary">操作按钮</Button>
  </CardFooter>
</Card>
```

#### 自定义类名
```tsx
<Card className="mb-4">
  {/* 卡片内容 */}
</Card>

<CardHeader className="bg-aden-light-gray">
  {/* 头部内容 */}
</CardHeader>
```

#### 嵌套使用
```tsx
<Card>
  <CardHeader>
    <h3 className="text-xl font-title font-bold">服务项目</h3>
  </CardHeader>
  <CardBody>
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
      <Card>
        <CardBody>
          <h4 className="font-title font-bold mb-2">服务1</h4>
          <p className="text-aden-dark-gray font-body">服务描述...</p>
        </CardBody>
      </Card>
      <Card>
        <CardBody>
          <h4 className="font-title font-bold mb-2">服务2</h4>
          <p className="text-aden-dark-gray font-body">服务描述...</p>
        </CardBody>
      </Card>
    </div>
  </CardBody>
</Card>
```

## 国际化组件

### LanguageProvider 组件

#### 位置
`components/ui/LanguageProvider.tsx`

#### 功能
提供国际化上下文，管理语言切换和翻译内容。

#### 导入方式
```tsx
import { useLanguage } from '../components/ui/LanguageProvider';
```

#### 使用方式
```tsx
const { lang, setLang, translations } = useLanguage();

// 使用翻译内容
<h1>{translations?.home?.hero?.title || '默认标题'}</h1>

// 切换语言
<button onClick={() => setLang('en')}>切换到英语</button>
```

### LanguageSwitcher 组件

#### 位置
`components/ui/LanguageSwitcher.tsx`

#### 功能
提供语言切换下拉菜单。

#### 导入方式
```tsx
import LanguageSwitcher from '../components/ui/LanguageSwitcher';
```

#### 使用方式
```tsx
<LanguageSwitcher 
  currentLang={lang} 
  onLanguageChange={setLang} 
/>
```

## 样式规范

### 颜色使用
- 使用 `aden-*` 前缀的颜色类，如：
  - `bg-aden-orange` - 主橙色
  - `bg-aden-dark-blue` - 深蓝色
  - `text-aden-dark-gray` - 深灰色文本
  - `border-aden-mid-grey` - 中灰色边框
  - `bg-aden-light-gray` - 浅灰色背景

### 字体使用
- `font-body` - 正文字体
- `font-title` - 标题字体
- `font-cn` - 中文页面字体

### 布局规范
- 使用 `container-content` 类进行页面布局
- 避免直接使用固定宽度，如 `w-[1180px]`

## 最佳实践

1. **优先使用统一组件**：尽量使用 `components/ui` 目录下的统一组件，避免在页面中直接写样式

2. **保持组件纯净**：UI 组件应保持纯展示性，通过 props 接收数据，不直接访问业务逻辑

3. **遵循目录结构**：按照功能将组件分类到对应目录，保持代码结构清晰

4. **使用 Tailwind 类**：优先使用 Tailwind 类进行样式定义，避免使用行内 style

5. **国际化支持**：所有文案应通过 `translations` 对象获取，支持多语言

6. **响应式设计**：使用 Tailwind 的响应式类，确保在不同设备上的显示效果

7. **代码审查**：按照代码审查规则检查组件使用是否符合规范
