## 🎨 Section 组件使用指南

### 1. HeroSection - 页面头部

**用途：** 用于页面顶部的 Hero 区域，带标题和副标题

**基本用法：**
```tsx
import HeroSection from '../../components/sections/HeroSection';

export default function MyPage() {
  return (
    <HeroSection
      title="End-to-End Intelligent Solutions."
      subtitle="From design to long-term ESG operations..."
    />
  );
}
```

**高级用法：**
```tsx
<HeroSection
  title="Our Solutions"
  subtitle="Tailored solutions for every industry"
  backgroundImage="/images/hero.jpg"
  overlay={true}
  align="center"
  className="py-32"
/>
```

**Props 说明：**
- `title`（必填）：主标题
- `subtitle`（可选）：副标题
- `backgroundImage`（可选）：背景图片 URL
- `overlay`（可选）：是否添加黑色遮罩（默认 true）
- `align`（可选）：对齐方式 `'left' | 'center' | 'right'`（默认 left）
- `className`（可选）：额外样式类

---

### 2. CTASection - 行动号召区域

**用途：** 用于页面底部的 CTA 区域，引导用户联系

**基本用法：**
```tsx
import CTASection from '../../components/sections/CTASection';

export default function MyPage() {
  return (
    <CTASection
      title="Ready to transform your project?"
      body="Our team of experts is ready to help you..."
      ctaText="Get in Touch"
      ctaLink="/contact"
    />
  );
}
```

**变体用法：**
```tsx
{/* 深色背景（默认） */}
<CTASection
  title="Contact Us"
  variant="dark"
/>

{/* 浅色背景 */}
<CTASection
  title="Learn More"
  variant="light"
/>
```

**Props 说明：**
- `title`（必填）：主标题
- `body`（可选）：描述文字
- `ctaText`（可选）：按钮文字（默认 "Get in Touch"）
- `ctaLink`（可选）：按钮链接（默认 "/contact"）
- `variant`（可选）：变体 `'dark' | 'light'`（默认 dark）
- `className`（可选）：额外样式类

---

### 3. ContentSection - 通用内容区域

**用途：** 用于展示通用内容，支持标题、副标题和自定义内容

**基本用法：**
```tsx
import ContentSection from '../../components/sections/ContentSection';

export default function MyPage() {
  return (
    <ContentSection
      title="Our Services"
      subtitle="Comprehensive solutions for your needs"
    >
      <p>Your content here...</p>
    </ContentSection>
  );
}
```

**变体用法：**
```tsx
{/* 白色背景（默认） */}
<ContentSection title="..." variant="white">
  ...
</ContentSection>

{/* 灰色背景 */}
<ContentSection title="..." variant="gray">
  ...
</ContentSection>

{/* 限制内容宽度 */}
<ContentSection maxWidth="prose">
  ...
</ContentSection>
```

**Props 说明：**
- `title`（可选）：区域标题
- `subtitle`（可选）：区域副标题
- `children`（必填）：子内容
- `variant`（可选）：背景变体 `'white' | 'gray'`（默认 white）
- `maxWidth`（可选）：最大宽度 `'default' | 'prose' | 'full'`（默认 default）
- `className`（可选）：额外样式类

## 🎬 Motion 组件使用指南

### 1. FadeIn - 淡入动画

**用途：** 元素进入视口时淡入

**基本用法：**
```tsx
import FadeIn from '../../components/motion/FadeIn';

export default function MyPage() {
  return (
    <FadeIn>
      <h2>Our Services</h2>
      <p>Content...</p>
    </FadeIn>
  );
}
```

**高级用法：**
```tsx
<FadeIn
  delay={0.2}
  duration={0.8}
  direction="up"
  threshold={0.2}
  once={false}
>
  <ContentSection title="...">...</ContentSection>
</FadeIn>
```

**Props 说明：**
- `children`（必填）：子元素
- `className`（可选）：额外样式类
- `delay`（可选）：延迟时间（秒，默认 0）
- `duration`（可选）：持续时间（秒，默认 0.6）
- `direction`（可选）：进入方向 `'up' | 'down' | 'left' | 'right' | 'none'`（默认 up）
- `threshold`（可选）：触发阈值 0-1（默认 0.1）
- `once`（可选）：是否只触发一次（默认 true）

---

### 2. SlideIn - 滑入动画

**用途：** 元素从侧面滑入

**基本用法：**
```tsx
import SlideIn from '../../components/motion/SlideIn';

<SlideIn direction="left">
  <Card>...</Card>
</SlideIn>
```

**Props 说明：**
- `direction`（可选）：滑入方向 `'left' | 'right'`（默认 left）
- `distance`（可选）：滑入距离（像素，默认 50）
- 其他 Props 同 FadeIn

---

### 3. ScaleIn - 缩放淡入动画

**用途：** 元素从小到大淡入

**基本用法：**
```tsx
import ScaleIn from '../../components/motion/ScaleIn';

<ScaleIn scale={0.8}>
  <div className="bg-white p-6">...</div>
</ScaleIn>
```

**Props 说明：**
- `scale`（可选）：初始缩放比例（默认 0.9）
- 其他 Props 同 FadeIn

---

### 4. StaggerGrid - 交错网格动画

**用途：** 用于网格布局，子元素依次出现

**基本用法：**
```tsx
import StaggerGrid from '../../components/motion/StaggerGrid';

<StaggerGrid staggerDelay={0.1}>
  {items.map((item, index) => (
    <Card key={index}>{item}</Card>
  ))}
</StaggerGrid>
```

**Props 说明：**
- `children`（必填）：子元素数组
- `staggerDelay`（可选）：每个子元素的延迟（秒，默认 0.1）
- `direction`（可选）：进入方向 `'up' | 'down' | 'none'`（默认 up）
- 其他 Props 同 FadeIn

## 📋 组合使用示例

### 示例 1：标准页面结构

```tsx

import ContentSection from '../../components/sections/ContentSection';
import CTASection from '../../components/sections/CTASection';
import FadeIn from '../../components/motion/FadeIn';

  return (
    <div className="min-h-screen">
      {/* Hero Section */}

      {/* Content Section */}
      <FadeIn>
        <ContentSection
          title="What We Offer"
          subtitle="Tailored services for every need"
          variant="white"
        >
          <div className="grid grid-cols-1 md:grid-cols-3 gap-8">
```

## ✅ 使用规范

### 推荐做法

✅ **使用封装的组件：**
```tsx
<HeroSection title="..." />
<ContentSection title="...">...</ContentSection>
<FadeIn>...</FadeIn>
```

✅ **组合使用：**
```tsx
<FadeIn>
  <ContentSection title="...">
    <StaggerGrid>...</StaggerGrid>
  </ContentSection>
</FadeIn>
```

✅ **保持简洁：**
```tsx
// 简单场景直接用 FadeIn
<FadeIn>
  <p>Simple content</p>
</FadeIn>
```

---

### 不推荐做法

❌ **不要重复造轮子：**
```tsx
// 错误：自己写 Hero 样式
<section className="bg-aden-dark-blue text-white py-20 md:py-32">
  <div className="container-content">
    <h1 className="text-4xl md:text-5xl lg:text-6xl font-title font-bold mb-6">
      Our Services
    </h1>
  </div>
</section>

// 正确：使用 HeroSection
<HeroSection title="Our Services" />
```

❌ **不要过度使用动画：**
```tsx
// 错误：每个元素都用不同动画
<FadeIn>...</FadeIn>
<SlideIn>...</SlideIn>
<ScaleIn>...</ScaleIn>

// 正确：统一使用 FadeIn 或 StaggerGrid
<StaggerGrid>...</StaggerGrid>
```

❌ **不要在小组件上用动画：**
```tsx
// 错误：Button 上用动画
<FadeIn>
  <Button>Click me</Button>
</FadeIn>
```

---

## 🎯 最佳实践

1. **优先使用 Section 组件**：不要自己写 Section 样式
2. **Motion 组件用于 Section 级**：不要用于 Button、Input 等小组件
3. **保持动画一致**：整个网站使用相同的动画参数
4. **性能优先**：设置 `once={true}` 避免重复动画
5. **可访问性**：动画不影响功能，只是增强体验
