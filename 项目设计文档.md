# 旅游网站项目设计文档

## 1. 项目概述

### 1.1 项目背景
本项目旨在构建一个基于世界地图的旅游记录网站，用户可以通过微信登录后，在不同国家上传旅游照片，并通过地图可视化展示自己的旅行足迹。

### 1.2 项目目标
- 提供微信登录功能，简化用户注册登录流程
- 实现交互式世界地图，支持缩放和国家点击
- 允许用户在国家子页面上传旅游照片
- 通过颜色变化直观显示用户已上传照片的国家

## 2. 技术架构

### 2.1 架构概述
本项目采用前后端分离的架构设计，前端使用Next.js框架（Server Components 优先），后端使用Firebase服务。项目严格遵循前端治理规范，确保品牌一致性、可维护性和可扩展性。

| 编程语言 | TypeScript | 5.3.3 | 类型安全的JavaScript |

│   └── PhotoGallery.tsx  # 照片展示组件
└── Map/              # 地图相关组件
    ├── WorldMap.tsx  # 世界地图组件
    └── CountryLayer.tsx  # 国家图层组件

contexts/             # 上下文
└── UserContext.tsx   # 用户状态上下文

lib/                  # 工具函数和配置

├── map.ts            # 地图相关服务

#### 3.2.2 数据模型

**用户数据模型**：
```typescript
interface User {
  id: string;            // 用户唯一标识
  wechatId: string;      // 微信ID
  nickname: string;      // 微信昵称
  avatar: string;        // 微信头像
  countries: string[];   // 已上传照片的国家列表
  createdAt: timestamp;  // 创建时间
}
```

**照片数据模型**：
```typescript
interface Photo {
  id: string;            // 照片唯一标识
  userId: string;        // 所属用户ID
  country: string;       // 国家名称
  url: string;           // 照片存储URL
  description?: string;  // 照片描述（可选）
  createdAt: timestamp;  // 上传时间
}
```

**国家数据模型**：
```typescript
interface Country {
  code: string;          // 国家代码
  name: string;          // 国家名称
  hasPhotos: boolean;    // 是否有用户照片
}
```

  - 页面间距：section间距使用sizeMap（sm: py-12 md:py-16, md: py-16 md:py-20, lg: py-20 md:py-28）

### 5.2 页面详细设计

#### 5.2.1 主页（世界地图）
- **地图设计**：
  - 地图风格：使用OpenStreetMap的标准风格，浅色背景
  - 国家边界：1px实线，颜色为aden-mid-grey

  - 提示信息：悬停时显示国家名称，使用Card组件
- **导航栏**：
  - 样式：fixed定位，半透明背景（bg-aden-white/90），backdrop-blur
  - 内容：左侧品牌Logo，右侧用户头像/登录按钮
  - 响应式：移动端显示汉堡菜单，桌面端显示完整导航
- **登录弹窗**：
  - 样式：居中弹窗，使用Card组件，阴影效果
  - 内容：微信扫码区域，登录提示文本
  - 动画：FadeIn动画，从中心弹出

#### 5.2.2 国家子页面
- **页面布局**：
  - 顶部导航栏：包含国家国旗、名称（font-title）和返回按钮（使用Button组件）
  - 照片上传区域：居中显示，包含拖拽上传区域（虚线边框）和文件选择按钮（primary按钮）
  - 照片展示区域：响应式网格布局，根据屏幕尺寸调整列数
- **交互元素**：
  - 上传按钮：使用品牌橙色primary按钮
  - 照片卡片：使用Card组件，包含照片、上传时间和删除按钮
  - 返回按钮：使用outline按钮，左侧箭头图标

### 5.3 交互设计

#### 5.3.1 微交互
- **按钮反馈**：
  - 悬停：轻微缩放（scale-105），透明度变化
  - 点击：按下效果（scale-95）
  - 加载：显示加载动画
- **表单交互**：
  - 输入框：焦点时边框变为aden-digital-blue
  - 错误提示：红色文本，位于输入框下方
- **地图交互**：
  - 缩放：平滑动画，缩放级别变化时的过渡效果
  - 国家选择：点击时短暂高亮效果
- **上传交互**：
  - 拖拽：拖拽区域高亮，显示“释放以上传”提示
  - 进度条：实时显示上传进度，使用aden-orange填充

#### 5.3.2 用户流程
- **首次访问流程**：
  1. 进入网站 → 显示登录弹窗
  2. 微信扫码 → 登录成功
  3. 显示世界地图 → 可交互
- **照片上传流程**：
  1. 点击国家 → 进入子页面
  2. 选择照片（拖拽或文件选择）
  3. 显示上传进度 → 上传完成
  4. 照片显示在网格中 → 返回地图
  5. 地图中国家颜色变化 → 显示为已访问
- **浏览流程**：
  1. 缩放地图 → 查看不同区域
  2. 点击国家 → 浏览照片
  3. 返回地图 → 继续探索
